Temporary VIP Commands
Command Description Example
!addvip Adds VIP time to the specified user. If [Time] has been left blank, it will pick the default time that is set in the action. !addvip Razinbazin 1mo
!removevip Removes VIP time. If the removed time exceeds the time the user had left, it will un-vip them. So to completely remove someone from being a temporary VIP, you can just remove 100 years. !removevip saintclear 2w
!refreshvip A manual refresh and checks for expired VIPs. If any are expired, they will be removed. !refreshvip
Moderator Editable Commands (BE EXTREMELY CAREFUL WITH THESE)
Command Description Example
!comfirst This must be run for the system too register all of the commands. It will deactivate after first run !comfirst
!comlist Lists all of the commands. !comlist
!addcom This creates a new command with the bot. It is important to remember not to put the ! before the command name, as in the example. !addcom fart @%user% farts in @%targetUser%'s direction.
!editcom This will edit a command with new info. Do not put the ! before the command name, just like with adding commands. !editcom fart @%user% blows a MASSIVE fart in @%targetUser%'s direction.
!delcom BE EXTREMELY CAREFUL WITH THIS COMMAND. YOU CAN DESTROY THE BOT (not being dramatic). Deletes commands from the bot. !delcom fart
